易错一：

  • 字符串使用单引号、双引号、三引号作为定界符，也就是说’ss’这个也是字符串，不是字符
  • 字符串和元组属于不可变序列，不能通过下标的方式来修改其中的元素值，试图修改元组中元素的值时会抛出异常。
  • 余数与%右侧的运算数符号一致
  • 比较运算默认都是比字典序，不是比长度

易错二：

  • Python交互模式启动时，会对[-5, 256]区间的整数进行缓存。也就是说，如果多个变量的值相等且介于[-5, 256]区间内，那么这些变量共用同一个值的内存空间
  • 同一个程序中 或 交互模式下同一个语句中 的同值不同名变量会共用同一个内存空间
  • is用来测试两个对象是否是同一个，如果是则返回True，否则返回False。如果两个对象是同一个，二者具有相同的内存地址。就是比id(x) == id(y)
